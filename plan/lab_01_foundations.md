# Lab 01 — Foundations Brief

## Objective
Create a deterministic baseline runtime for Isaac Sim lab exercises with config loading, environment setup, core control loop, artifact generation, and repeatable checks.

## Required Deliverables
- Config files for default and development runs.
- Source modules for config parsing, setup, runtime loop, and artifact writing.
- Scripts to run the lab and validate repository structure.
- Unit and pipeline tests.
- Planning artifacts in `tasks/`.

## Exit Criteria
- `scripts/run_phase_checks.sh` passes.
- Unit tests pass.
- Core run script produces expected artifacts.
