# Lab 03 — Perception Brief

## Objective
Implement a deterministic perception scaffold that simulates sensor output, extracts features, and writes reproducible artifacts.

## Required Deliverables
- Perception config and sensor placeholder models.
- Modules for setup, sensor simulation, feature extraction, and output logging.
- Run and validation scripts.
- Unit/pipeline tests.

## Exit Criteria
- `scripts/run_phase_checks.sh` passes.
- Fixed-seed runs produce stable feature output.
