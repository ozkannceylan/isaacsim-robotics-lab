# Lab 02 Navigation

This lab implements the deterministic waypoint-planning scaffold described in `plan/lab_02_navigation.md`.

## Run

```bash
bash lab_02_navigation/scripts/run_lab.sh
```

## Run all checks

```bash
bash lab_02_navigation/scripts/run_phase_checks.sh
```

## Run tests only

```bash
python3 -m unittest discover -s lab_02_navigation/tests -p 'test_*.py'
```
